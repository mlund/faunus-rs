## Claude Code

A `faunus-yaml` skill is available to help create and modify YAML input files.
It is automatically triggered when you ask about setting up systems, configuring
energy terms, MC moves, analysis, or working with state files, e.g.:

~~~
Set up an NPT simulation of 256 SPC/E water molecules with Ewald summation
~~~

## Examples

Command               | Description
--------------------- | -----------------------------------------------
`scripts/twobody.sh`  | Free energy between two rigid multipolar bodies
